## The Prompt

You are a food copywriter and menu engineer specializing in restaurant revenue optimization. Rewrite and optimize the menu copy below for maximum order value, digital delivery platform performance, and full regulatory compliance. Every output must be publish-ready.

**RESTAURANT DETAILS:**
- Restaurant name: [NAME]
- Cuisine type: [E.g., "Modern Mexican," "Southern BBQ," "Japanese ramen"]
- Location (state): [STATE] — used for alcohol and labeling rules
- Chain size: [Independent / 2–19 locations / 20+ locations]
- Alcohol service: [Full bar / Beer & wine / None]
- Online ordering platforms active: [DoorDash / Uber Eats / Grubhub / Toast / all / none]
- Top 3 high-margin items you want to sell more of: [ITEM 1, ITEM 2, ITEM 3]
- Any allergen-free or special diet options: [Gluten-free / Vegan / Nut-free / None]

**CURRENT MENU (paste your current menu items and descriptions here):**
[PASTE MENU — item names, current descriptions, prices]

**GENERATE ALL OF THE FOLLOWING:**

**1. Rewritten Menu Section Copy**
For each item provided:
- Rewrite description using sensory language (flavor, texture, aroma, temperature, origin)
- 20–40 words per item for print/in-store; 140–150 characters for DoorDash/Uber Eats/Grubhub
- Apply menu engineering principles: anchor high-margin items, use decoy pricing framing if 3+ tiers
- Label high-margin items as "Chef's Pick" or "House Favorite" (NOT "Best in [city]" without attribution)
- No health claims without FDA substantiation: do NOT use "healthy," "low-calorie," "guilt-free," "detox," "superfood," "immune-boosting" unless backed by approved nutrient content claim (21 CFR 101.13)
- Allergen callouts: bold or parenthetical notation for Big 9 allergens (milk, eggs, fish, shellfish, tree nuts, peanuts, wheat, soy, sesame) if present
- "Locally sourced" or "farm-to-table" only if verifiable sourcing is stated in the restaurant details

**2. FDA Menu Labeling Compliance Guide**
Based on chain size = [CHAIN SIZE]:
- 20+ locations: FDA 21 CFR 101.11 requires calorie disclosure on menus and menu boards. Provide: (a) calorie display format ("480 Cal" next to item name), (b) required statement: "2,000 calories a day is used for general nutrition advice, but calorie needs vary," (c) placement rules
- 2–19 locations: not federally required; state laws vary (CA, NYC, Philadelphia have independent rules). Flag if location = CA or NY.
- Independent: voluntary; recommend for competitive positioning if avg check > $15
Provide a template implementation note for the menu design team.

**3. Alcohol Menu Copy (only if alcohol = Yes)**
TABC / State ABC compliance:
- No "drink free," "unlimited drinks," "all you can drink" language (promotes irresponsible consumption — TABC §11.38, state equivalents)
- No language targeting minors; no cartoon imagery for cocktail descriptions
- No claiming intoxicating effects as a selling point ("get you drunk fast," "strongest drink in town")
- Age-gate language for digital menus: "Must be 21+ to order alcohol. Please drink responsibly."
- Cocktail descriptions: flavor profile, base spirit, garnish — no claims about effect
- Happy hour compliance note: some states (TX, WA) restrict "happy hour" language or require specific formats — flag based on [STATE]
Write 5 cocktail/drink descriptions in compliant format.

**4. Online Delivery Platform Menu Copy**
For each platform active:
- **DoorDash:** Item name (35 chars max), description (140 chars max), dietary tags (Vegetarian/Vegan/Gluten-Free)
- **Uber Eats:** Item name (55 chars max), description (150 chars max)
- **Grubhub:** Item name (no hard limit, 40 recommended), description (255 chars max)
Rewrite the top 10 items from the current menu in each platform's format. Prioritize the 3 high-margin items given above. Flag items where current description exceeds platform limits.

**5. Limited-Time Item (LTO) Launch Copy**
For a seasonal or promotional item:
- Item name and 30-word description (print)
- 140-char DoorDash description
- In-store table tent headline + 2-sentence body
- Social post caption (150 words, 15 hashtags)
- Email blast subject line + 3-sentence preview
- Timeline: "Available [START DATE] through [END DATE] while supplies last" — exact dates required (FTC requires specific end dates for limited-time offers)

**6. Allergen Disclaimer Template**
FDA FSMA Big 9 Allergens (milk, eggs, fish, shellfish, tree nuts, peanuts, wheat, soy, sesame):
- Full menu disclaimer (for print menu footer and website): "We handle ingredients containing milk, eggs, tree nuts, peanuts, wheat, soy, sesame, fish, and shellfish in our kitchen. Cross-contact is possible. Please inform your server of any allergies before ordering."
- Server script for allergy inquiries: "Thank you for letting me know. Our kitchen does handle [allergen]. I can connect you with our manager to discuss your options before you order. Can I get your server card flagged?"
- Online ordering allergy notice for each platform
