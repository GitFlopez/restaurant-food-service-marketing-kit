# Restaurant & Food Service Marketing Kit

Complete marketing system for restaurants, cafes, food trucks, catering companies, and food service businesses. 4 prompts covering grand opening, menu copywriting, customer retention, and local SEO + paid ads — with FDA, FTC, TABC, and TCPA compliance built in at generation time.

## What's Included

| File | Description |
|------|-------------|
| `SKILL.md` | Full skill documentation and pricing |
| `prompts/01-grand-opening-acquisition.md` | Grand opening + new customer acquisition kit |
| `prompts/02-menu-copywriting-optimization.md` | Menu copy, FDA labeling, delivery platform optimization |
| `prompts/03-customer-retention-loyalty.md` | Email/SMS retention, loyalty, review management |
| `prompts/04-local-seo-social-paid-ads.md` | Local SEO, social calendar, Google + Meta ads, influencer outreach |
| `examples/copper-skillet-henderson/` | Complete worked example — Henderson NV farm-to-table diner |

## Quick Start

1. Open any prompt file
2. Fill in the `[BRACKET]` fields with your restaurant's details
3. Paste into Claude (Sonnet or Opus recommended)
4. Outputs are publish-ready — no additional editing required for compliance

## Compliance Coverage

- **FDA Menu Labeling Rule (21 CFR 101.11):** Calorie disclosure guidance by chain size
- **FDA Big 9 Allergens (FSMA):** Allergen disclosure in all menu copy and server scripts
- **FTC 2023 Endorsement Guides:** No conditional review requests, no incentivized endorsements, influencer disclosure brief included
- **TABC / State ABC:** No intoxication promotion, age-gate language for digital alcohol menus, happy hour compliance by state
- **TCPA:** Double opt-in, opt-out in every SMS, sender ID required
- **CAN-SPAM:** Unsubscribe + physical address in all email templates
- **State Health Permit Display:** Nevada, California, Texas guidance included; general guidance for other states

## Example Use Case

**The Copper Skillet** — farm-to-table American diner, Henderson NV. See `examples/copper-skillet-henderson/` for complete output across all 4 prompts including Google RSA ads, schema FAQ JSON-LD, 5-email retention sequence, and press release.
